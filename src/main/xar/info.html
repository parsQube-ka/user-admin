eXist-db LGPL Eine NoSQL-Dokumentendatenbank und Anwendungsplattform.
jQuery MIT JQuery ist eine schnelle, kleine und funktionsreiche JavaScript Library. Es macht HTML-Dokument-Traversal und Manipulation, Event-Handling, Animation und Ajax viel einfacher mit einer easy-to-use API, die über eine Vielzahl von Browsern funktioniert. Mit einer Kombination aus Vielseitigkeit und Erweiterbarkeit hat jQuery die Art und Weise verändert, dass Millionen von Menschen JavaScript schreiben.
DataTables MIT DataTables ist ein Plug-In für die jQuery Javascript Library. Es ist ein sehr flexibles Werkzeug, basierend auf den Grundlagen der progressiven Verbesserung, und fügt erweiterte Interaktion Kontrollen zu jeder HTML-Tabelle.
